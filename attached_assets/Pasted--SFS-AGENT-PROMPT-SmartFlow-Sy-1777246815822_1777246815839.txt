STEP 8 — ANIMATED GOLD PARTICLE BACKGROUND (sfs-bg.js)
--------------------------------------------------------------------
Self-injecting — just add the <script> tag, no init needed.

What it does:
  • Creates a fixed <canvas id="sfs-bg-canvas"> behind all content
  • Draws 45 drifting gold nodes connected by fading gold lines
  • Draws 60 twinkling gold sparkles (breathing opacity)
  • Bounces nodes off edges, pauses when tab is hidden (battery saving)
  • On mobile (≤768px) halves node and sparkle count automatically

Configuration (edit CONFIG object at top of sfs-bg.js):
  nodeCount:          45     ← denser = more connections
  connectionDistance: 160    ← max line length in px
  nodeSpeed:          0.28   ← drift speed
  nodeRadius:         2.2    ← dot size
  goldAlpha:          0.14   ← line opacity
  sparkleCount:       60     ← twinkling stars
  sparkleMaxRadius:   1.6    ← max star size
  sparkleSpeed:       0.008  ← twinkle pulse speed
  mobileBreakpoint:   768    ← px below which to halve counts
  pauseOnHide:        true   ← pause when tab not visible

Canvas sits at z-index: -1 (fixed, full viewport, pointer-events:none).
All other content auto-gets z-index: 1 via CSS rule in sfs-ui.css.

Particle colour: rgba(255,215,0, alpha)  — always gold.
Sparkle colour:  rgba(255,215,0, 0–0.45) — pulsing.
